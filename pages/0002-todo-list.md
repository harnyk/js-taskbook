# Список задач и функции для его отображения

В этом задании вам предстоит написать функции для обработки списка задач, используя JavaScript. Каждый уровень сложности включает в себя новую функциональность, основываясь на предыдущих результатах.

**Данные:**

```javascript
const tasks = [
    {
        id: 1,
        description: 'Пойти в магазин',
        completed: false,
        dueDate: new Date('2024-01-20'),
    },
    {
        id: 2,
        description: 'Выгулять собаку',
        completed: true,
        dueDate: new Date('2024-01-18'),
    },
    {
        id: 3,
        description: 'Приготовить ужин',
        completed: false,
        dueDate: new Date('2024-01-19'),
    },
    {
        id: 4,
        description: 'Почитать книгу',
        completed: true,
        dueDate: new Date('2024-01-21'),
    },
    {
        id: 5,
        description: 'Заняться спортом',
        completed: false,
        dueDate: new Date('2024-01-22'),
    },
    {
        id: 6,
        description: 'Позвонить другу',
        completed: false,
        dueDate: new Date('2024-01-18'),
    },
    {
        id: 7,
        description: 'Убраться в комнате',
        completed: true,
        dueDate: new Date('2024-01-20'),
    },
];
```

**Уровень 1: Основной**

-   Напишите функцию `formatTasks`, которая использует `map` и `join`, чтобы создать одну большую строку, отображающую все задачи со статусом выполнения.
-   **Результат:** Функция возвращает строку в формате:

```
1. Пойти в магазин - Не выполнено
2. Выгулять собаку - Выполнено
3. Приготовить ужин - Не выполнено
и т.д....
```

**Уровень 2: Фильтрация**

-   Измените `formatTasks`, добавив фильтрацию для отображения только невыполненных задач.
-   **Результат:** Функция возвращает строку только с невыполненными задачами.

**Уровень 3: Сортировка по описанию**

-   Модифицируйте `formatTasks`, добавив сортировку задач по алфавиту (по полю `description`).
-   **Результат:** Функция возвращает отсортированный по описанию список задач.

**Уровень 4: Сортировка по дате и отображение даты**

-   Добавьте в `formatTasks` отображение даты в произвольном формате.
-   Модифицируйте `formatTasks`, добавив сортировку задач по дате выполнения (`dueDate`).
-   **Результат:** Функция возвращает отсортированный по дате список задач, где каждая задача отображается в формате "1. Пойти в магазин - Невыполнено - 20.01.2024".

**Изучаемые темы:**

-   Методы массивов (`map`, `filter`, `sort`, `join`).
-   Работа с объектами и массивами объектов.
-   Форматирование и вывод дат.
-   Строковые операции и шаблонные строки.
-   Условные выражения и сортировка данных.

Это задание поможет развить навыки работы с различными методами обработки массивов и объектов, а также улучшит понимание управления данными и их представления в JavaScript.
